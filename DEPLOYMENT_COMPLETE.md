## 🔐 Security Checklist

Before deploying, remember:

- ✅ **Generate strong SECRET_KEY** - Use: `python -c "import secrets; print(secrets.token_urlsafe(32))"`
- ✅ **Set unique database password** - Never reuse defaults
- ✅ **Store API keys securely** - Use platform's secure variable storage
- ✅ **Remove bootstrap admin credentials** - Delete after first login
- ⏳ **Rotate secrets monthly** - Plan for JWT secret rotation
- ⏳ **Enable monitoring alerts** - Configure after deployment
- ⏳ **Review security logs** - Check for suspicious activity
